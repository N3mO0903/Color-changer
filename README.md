# Simple Color Changer

## Description
This is a simple web project that changes the background color of the page. Users can type a color name (like "red") or a hex code (like "#ff0000") and change the page background. There is also a reset button to restore the original color.

## Features
1. Input color name or hex code.
2. Change background color.
3. Reset to default color.

## How to Run
1. Download the files.
2. Open `index.html` in a web browser.
3. Enter a color and click "Change Color".
4. Click "Reset" to restore the original background.
